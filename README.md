make sure your python version >= 3.9.6 (We never tested lower versions; You could just run the code and if nothing breaks, call it a day)


install requirements by running “python -m pip install -r requirements.txt”


run "python -m src.rag_service.api" in the root folder to start the server


run "python -m streamlit run streamlit_app.py" in the root folder to start the front end web app
